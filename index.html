Full Stack Developer with Ruby (HTML, CSS, Ruby and Ruby on rails)

Intro to Full Stack Ruby Developer merupakan tema pelatihan di mana para peserta akan mempelajari dasar-dasar dari 2 bahasa pemrograman dan 1 framework yaitu: HTML & CSS, Ruby, dan Ruby on Rails. Ruby banyak digunakan oleh perusahaan-perusahaan besar seperti Gojek, crunchbase, dan airbnb. Pelatihan ini akan dilaksanakan selama 5 minggu dimana peserta akan mempelajari seluruh ilmu penting untuk mengembangkan website dan aplikasi berbasis web dengan praktik langsung di platform online coding Progate, mengerjakan berbagai proyek seru, sesi live, dan didampingi oleh Instruktur berpengalaman secara daring.
